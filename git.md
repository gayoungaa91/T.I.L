## 나 보려고 만든 Git 명령어 정리

##### - 원격 저장소에서 변경사항을 가져와 현재 브랜치에 합치기

#### `git pull origin master`

##### - 원격 저장소에 잘못 올라간 파일 삭제

#### `git rm --cached 파일/폴더명 `

##### - 파일 삭제

#### `git rm -f 파일/폴더명 `
